## 세부 태스크

### TASK-5.2-1 — 2D 횡스크롤 풀밭 배경
**파일**: `webview/renderer.js`, `extension.js` (HTML)

- Canvas 전체에 하늘 + 땅 레이어를 그리는 `drawBackground()` 함수 구현
- **하늘**: 윗 절반, 연한 파랑 계열 단색 (`#87CEEB` → 흰 구름 픽셀 몇 개)
- **땅**: 아랫 절반, 2레이어
  - 먼 배경 언덕: 어두운 초록 실루엣 (2~3픽셀 높이)
  - 지면: 밝은 초록 풀밭 줄, 아래로 갈수록 짙어지는 흙색
- **풀**: 지면 경계선에 ↑↑↑ 형태 풀 픽셀 반복 배치 (2~3px 높이)
- 배경은 현재 고정 (스크롤 없음) — 이동 기능은 추후 추가
- 몬스터/알은 지면 위에 배치 (`y = groundY - spriteHeight`)
- `drawSprite` 호출 전에 `drawBackground()` 먼저 실행 (레이어 순서)

### TASK-5.2-2 — 알 크기 배경 맞춤 조정
**파일**: `webview/renderer.js`

- 현재 `STAGE_SCALE[0] = 8` (16px × 8 = 128px) → 배경이 생기면 너무 큼
- 배경 지면 위에서 자연스러운 크기로 스케일 조정
  - 목표: 알이 캔버스 높이의 약 30~40% 수준
- 알 y 좌표를 `groundY` 기준으로 계산해 지면 위에 착지하도록 배치

### TASK-5.2-3 — 알 대기 애니메이션 개선 (좌우 흔들림)
**파일**: `src/sprites/egg.js`, `webview/renderer.js`

- **제거**: FRAME_B의 발 그림자 및 1px 이동 방식 삭제
- **변경**: 알 전체가 좌우로 미세하게 기울어지는 다중 프레임 애니메이션
  - 프레임 구성 (4~6프레임으로 부드러운 루프):
    - F0: 중앙 (기본 자세)
    - F1: 살짝 오른쪽으로 기운 형태
    - F2: 조금 더 오른쪽
    - F3: 중앙 복귀
    - F4: 살짝 왼쪽으로 기운 형태
    - F5: 조금 더 왼쪽 → F0으로 루프
  - 기울임 표현: 알 픽셀 행 전체를 ±1px 시프트하거나, 상·하단 픽셀만 조금씩 이동
  - 프레임 간격: 현재 800ms → **200ms** (전체 한 사이클 ~1.2초)
  - `FRAME_INTERVAL`을 스테이지별로 다르게 설정하거나, 알 전용 상수로 분리

### TASK-5.2-4 — 알 스프라이트 해상도 업
**파일**: `src/sprites/egg.js`

- 현재 16×16 배열 → **24×24** 또는 **20×20** 픽셀 배열로 재설계
- 픽셀 수 증가로 다음 디테일 추가 가능:
  - 알 표면 곡면 그라데이션 (밝은 면 / 어두운 면 / 하이라이트)
  - 줄무늬 대각선 더 자연스럽게 표현
  - 알 하단 그림자 픽셀 (지면 접촉감)
- 팔레트 기존 6색 유지 또는 1~2색 추가 가능
- `STAGE_SCALE[0]` 값은 TASK-5.2-2에서 조정된 값 사용
