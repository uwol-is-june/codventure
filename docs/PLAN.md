# pixel_cat — 전체 로드맵

## v0.1 — MVP

**목표**: 고양이가 살아있다는 느낌. 기본 상호작용 가능.

### 애니메이션
- [ ] 걷기 (walk) — 기본 idle 상태에서 좌우로 걷는 루프
- [ ] 앉기 (sit) — 정지 후 앉는 자세
- [ ] 그루밍 (groom) — `/pet` 커맨드 이후 발로 얼굴 닦는 동작
- [ ] 잠자기 (sleep) — `/sleep` 커맨드 이후 눈 감고 Zzz 표시
- [ ] 먹기 (eat) — `/food` 커맨드 이후 먹는 동작

### 커맨드
- [ ] `pixel-cat.food` — 고양이에게 먹이 주기
- [ ] `pixel-cat.pet` — 고양이 쓰다듬기
- [ ] `pixel-cat.sleep` — 고양이 재우기
- [ ] `pixel-cat.open` — 고양이 패널 열기

### 인프라
- [ ] 상태바 기본 표시
- [ ] Webview 패널 열기/닫기
- [ ] Canvas 렌더링 루프
- [ ] VSCode ↔ Webview 메시지 통신

## v0.2 — 퀄리티 & 지속성

**목표**: 더 예쁘고, 껐다 켜도 기억한다.

### 스프라이트 개선
- [ ] 16×16 픽셀 기준 정밀한 고양이 스프라이트 설계
- [ ] 애니메이션 프레임 수 증가 (각 동작 4~8프레임)
- [ ] 색상 팔레트 정의 (몸통/줄무늬/눈/코)

### 상태 영속성
- [ ] `ExtensionContext.globalState`로 고양이 상태 저장
- [ ] VSCode 재시작 후에도 마지막 상태 복원
- [ ] 배고픔(hunger) / 행복도(happiness) 내부 스탯 추가
- [ ] 시간 경과에 따른 스탯 자연 감소

### 기타
